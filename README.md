### Vue Applications

#### Developer (Vue)

Application can be run from both a Docker environment or Node locally (NVM).

##### Using Node

```sh
$ yarn install
```

```sh
$ yarn serve
```

##### Using Docker

```sh
# uncomment - '8080:8080'
# uncomment - ./Library/developer-vue:/usr/share/nginx/html:delegated
$ docker-compose up
```
